# Mini CRM Messaging System

Um sistema de gestão de campanhas de mensagens construído com Laravel e React.

## Português do Brasil

### 🚀 Funcionalidades (planejadas)

- Gestão de contatos
- Criação de campanhas
- Envio de mensagens com filas
- Rastreamento de status de entrega

### 🧱 Tecnologias

- Laravel (API)
- React (Frontend)
- MySQL
- Redis (filas)
